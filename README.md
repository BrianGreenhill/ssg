# Static Site Generator

Generate static site from markdown files. Use [Hugo](https://gohugo.io/) instead of this.

## Usage

```bash
# generate static site
ssg generate

# watch for changes
ssg watch
```

## Installation

```bash
go install github.com/briangreenhill/ssg@latest
```

## Features

- [x] Template rendering
- [x] Configuration
- [x] Watch mode
